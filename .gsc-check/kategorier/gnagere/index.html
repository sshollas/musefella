Gnagere — mus og rotter

Komplett oversikt over mus og rotter i norske hjem. Identifisering, forskjeller, bekjempelse og guider.

🐭
Mus

Husmus er vanligst om høsten og vinteren. Handle raskt — de formerer seg fort.

Farenivå: middels
🐀
Rotter

Rotter er mer alvorlig enn mus. Utendørs rotteangrep — kontakt kommunen.

Farenivå: høy
